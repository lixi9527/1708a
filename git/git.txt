ls  list  列表
作用：打印当前目录下的文件列表
一般我们会利用这个指令查看当前文件目录下的内容，之后在确定要进哪个文件或文件夹
两个参数  分别是a参数 和 |  参数
写法： ls  -a  a参数是用来显示全部文件信息的

ls  -l     l 参数是用来显示各个文件夹和文件的详细信息的

mkdir 文件夹1  文件夹2 ，，，，
可以创建层级文件夹  -p

删除文件夹 
rmdir  remove  directory
用法：rmdir 要删除的文件夹名称
rmdir只能删除空文件夹，如果文件夹有内容，就不能删除

创建文件的命令 
touch 
用法： touch 要创建的文件名（要包含后缀名）
可以批量创建文件 touch文件名1  文件名2 ，，，

删除文件和文件夹   rm  remove
1，  rm 文件名/文件夹名
2， rm    rm -r 文件夹名  表示删除当前文件夹 并递归删除文件夹中所有的文件  一般不建议使用这个命令删除，因为这个命令删除的文件夹和文件是不能恢复的
3，rm -f 文件名/文件夹名  表示删除当前文件夹的时候，即使有报错也不显示
